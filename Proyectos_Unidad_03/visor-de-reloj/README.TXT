Projecto: visor-de-reloj
Autores: David Barnes and Michael Kolling

Este proyecto es parte del material para el libro

   Programación Orientada a Objetos con Java
   Una Introducción Práctica usando BlueJ
   Third edition
   David J. Barnes and Michael Kolling
   Pearson Educacion, 2007

Es discutido en el capítulo 3.  

Para usar este proyecto, cree una instancia de la clase VisorDeReloj y deje
la ventana de inspección abierta. Con la ventana del inspector abierta llame
a los métodos del objeto, tales como ticTac y ponerEnHora. Observe el
"cadVisor" en el inspector.

El campo cadVisor simula el dispositivo real del reloj. Si este fuera un
programa que corre en un reloj real, cambiaría el hardware real, físico
del visor en vez de esta cadena de caracteres. Lo demás sería, en general,
lo mismo.

Nota: Etse proyecto no implementa una adecuada gestión de errores.
