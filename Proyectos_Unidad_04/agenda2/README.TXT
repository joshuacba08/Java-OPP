Project: agenda2
Authors: David J. Barnes and Michael Kolling

Este proyecto provee una implementación de una agenda simple.

Este proyecto es parte del material suplmentario Java 5 para 
el capítulo 4 del libro

   Programación Orientada a Objetos con Java
   Una Introducción Práctica usando BlueJ
   Third edition
   David J. Barnes and Michael Kolling
   Pearson Educacion, 2007 

   
Para iniciar este proyecto, cree un objeto Agenda e invoque sus métodos.
